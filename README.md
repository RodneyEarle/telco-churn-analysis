# Telco Customer Churn Analysis

This project analyzes customer churn behavior for a telecommunications company using SQL and Tableau.

## Tools Used
PostgreSQL
SQL
Tableau

## Business Questions
- What percentage of customers churn?
- Which contract types have the highest churn?
- How does customer tenure affect churn?
- Which internet services experience the most churn?

## Key Insights
- Month-to-month contracts have the highest churn rate (~42%)
- Customers with tenure under 12 months churn the most (~47%)
- Fiber optic customers churn significantly more than DSL customers
- Long-term contracts significantly reduce churn risk
